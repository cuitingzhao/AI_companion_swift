# 「点点：你的目标搭子」隐私政策

**最后更新日期：2025年12月2日**

上海盖娅惠众网络科技有限公司（以下简称"我们"）非常重视您的隐私。本隐私政策说明了我们如何收集、使用、存储和保护您在使用"点点：你的目标搭子"应用（以下简称"本应用"）时的个人信息。

---

## 1. 信息收集

### 1.1 您主动提供的信息

| 信息类型 | 用途 | 是否必需 |
|---------|------|---------|
| 手机号码 | 账户注册与登录验证 | 是 |
| 昵称 | 个性化称呼 | 是 |
| 出生日期 | 八字计算与性格分析 | 是 |
| 出生时间 | 八字计算与性格分析 | 是 |
| 出生地点 | 八字计算（时区校准） | 是 |
| 性别 | 个性化服务 | 是 |
| 当前城市 | 本地化运势服务 | 否 |
| 聊天消息 | AI 对话服务 | 是（使用时） |
| 目标与任务 | 目标追踪服务 | 否 |
| 图片 | 聊天功能 | 否 |

### 1.2 自动收集的信息

| 信息类型 | 用途 |
|---------|------|
| 设备标识符 | 推送通知、安全验证 |
| 设备型号 | 服务优化 |
| 操作系统版本 | 兼容性保障 |
| 应用版本 | 服务支持 |
| IP 地址 | 安全防护、地区服务 |
| 使用日志 | 服务改进、问题排查 |

### 1.3 第三方服务收集的信息

本应用使用以下第三方服务，这些服务可能收集额外信息：

- **Apple App Store**：订阅管理、支付处理
- **Apple Push Notification Service (APNs)**：推送通知
- **[云服务提供商]**：数据存储

## 3. 信息共享

我们不会出售您的个人信息。我们仅在以下情况下共享您的信息：

### 3.1 服务提供商
我们可能与帮助我们运营服务的第三方服务提供商共享信息，包括：
- 云存储服务
- 支付处理服务
- 推送通知服务

这些服务提供商仅能在为我们提供服务的范围内访问您的信息，并有义务保护您的信息。

### 3.2 法律要求
在以下情况下，我们可能披露您的信息：
- 遵守法律法规或法院命令
- 保护我们的合法权益
- 防止欺诈或其他非法活动
- 保护用户或公众的安全

### 3.3 业务转让
如果我们进行合并、收购或资产出售，您的信息可能作为交易的一部分被转让。我们会在此类交易发生前通知您。
